### 원칙 1️⃣: 시총 필터링 - 5천억 ~ 10조 범위 내 투자

**근거**:
- 대손 거래 80%가 시총 5천억 미만에서 발생 (4건, -1,386,938원)
- 대박 거래 50%가 1조~3조 구간에서 발생 (2건, 449,539원)
- 초소형주(5천억 미만)는 변동성 극심 + 유동성 리스크

**실행 규칙**:
- ✅ **최우선 타겟**: 시총 1조 ~ 3조
- ✅ **차선 타겟**: 시총 5천억 ~ 1조, 3조 ~ 10조
- ❌ **절대 회피**: 시총 5천억 미만 (대손 확률 80%)

**체크 방법**:
```

- [ ] 종목 시가총액이 5천억 이상인가?
- [ ] 10조 이하인가?
- [ ] 가능하면 1조~3조 구간인가?
```

- ⚠️ **주의**: 업종보다 시총/거래대금이 더 중요

### 원칙 5️⃣: 진입 전 테스트 매매 필수

**근거**:
- 호가창 실제 유동성 확인
- 슬리피지 및 체결 속도 검증
- 급등 후 급락 함정 회피

**실행 규칙**:
- ✅ **테스트 수량**: 목표 수량의 10% 이하
- ✅ **테스트 기준**: 즉시 체결 여부, 스프레드 확인
- ✅ **통과 조건**: 0.5% 이내 슬리피지, 3초 이내 체결
- ❌ **실패 시**: 즉시 철수, 해당 종목 당일 재진입 금지

**체크 방법**:
```
테스트 매매:
1. [ ] 목표 수량의 10% 시장가 매수
2. [ ] 체결가와 호가 차이 0.5% 이내인가?
3. [ ] 3초 이내 체결되었는가?
4. [ ] 호가창 매수/매도 물량이 충분한가?

통과 시 → 본격 진입
실패 시 → 즉시 청산 후 철수
```

### 원칙 6️⃣: 과도한 분할 매매 회피

**근거**:
- 한국피아이엠 대손: 31회 분할 매수 → -342,520원
- 대박 거래 평균: 17.2회 분할 매수/매도
- 과도한 분할은 평균 단가 상승 + 수수료 증가

**실행 규칙**:
- ✅ **적정 분할**: 12~17회 (매수/매도 각각)
- ⚠️ **주의 구간**: 18~20회
- ❌ **위험 구간**: 21회 이상 (대손 확률 증가)
- 💡 **원칙**: 수익 구간에서만 추가 매수, 손실 구간은 중단

**분할 매수 전략**:
```
1차 진입: 총 자금의 30% (테스트 후)
2차 진입: +1~2% 상승 시 30% 추가
3차 진입: +2~3% 상승 시 40% 추가
최대 3회 분할 원칙

분할 매도 전략:
1차 익절: +3% 도달 시 50% 청산
2차 익절: +5% 도달 시 30% 청산
3차 익절: +7% 또는 12:00 이후 전량 청산
최대 3회 분할 원칙
```

**체크 방법**:
```
진입 중 확인:
- [ ] 현재 분할 매수 횟수가 17회 이하인가?
- [ ] 평균 단가가 목표 범위 내인가?
- [ ] 손실 구간(-1% 이상)에서 추가 매수하지 않았는가?
```

### 원칙 7️⃣: 수익 보다가 꺾이면 1만원 남기고 청산

**근거**:
- 욕심 부리다 수익이 손실로 전환되는 경우 다수
- 소액이라도 수익 실현으로 심리적 안정
- 내일의 기회를 위한 자본 보존

**실행 규칙**:
- ✅ **수익 모니터링**: 1분봉 기준 하락 전환 시 즉시 대응
- ✅ **최소 수익 확보**: 1만원 이상 수익 시 꺾이는 순간 청산
- ✅ **손절 기준**: -2% 도달 시 무조건 손절 (예외 없음)
- 💡 **마인드**: "내일도 장은 있다. 오늘은 여기까지."

**청산 시나리오**:
```
시나리오 1: +50,000원 → +30,000원 하락
- 조치: 즉시 청산 (30,000원 수익 확보)

시나리오 2: +15,000원 → +5,000원 하락
- 조치: 즉시 청산 (5,000원 수익 확보)

시나리오 3: +10,000원 → +3,000원 하락
- 조치: 절반 청산 후 재관망, 0원 도달 전 전량 청산

시나리오 4: -5,000원 → -20,000원 (손절 -2%)
- 조치: 무조건 손절, 오늘 거래 종료
```

**체크 방법**:
```
청산 판단:
- [ ] 현재 수익이 최고점 대비 30% 이상 하락했는가?
- [ ] 1분봉 음봉이 2개 이상 연속인가?
- [ ] 수익이 1만원 이상 남아있는가?
- [ ] 손실이 -2%에 근접했는가?

하나라도 Yes → 즉시 청산
```
